# SERVICE EXAMPLES

### **Unarmed Security Officers**
Unarmed Security Officers conducting access control and physical security both on‑site and with mobile patrol vehicles. Officers are connected through satellite communications to a regional Security Operations Center that provides real‑time reporting to all stakeholders.

### **Armed Security Officers**
Armed Security Officers conducting access control and physical security both on‑site and with mobile patrol vehicles. Officers are connected through satellite communications to a regional Security Operations Center that provides real‑time reporting to all stakeholders.

### **DoD Security Control Center Operations**
Provide twenty‑four‑hour Security Control Center and Badging Operations, Security Administration, and Facility Support Services to a Department of Defense facility as a subcontractor. The 12‑acre facility consists of three large buildings that house office space, foundries, and compartmentalized secure areas.

### **Bureau of Prisons Support Services**
Provide a tailored, diversified total solution consisting of twenty‑four‑hour armed security forces, Total Facility Management, Total Maintenance, and Logistics Procurement to the Bureau of Prisons as a subcontractor. The facility consists of 355,000 square feet of building space on over 600 acres.
